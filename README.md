## Deskripsi Aplikasi

Aplikasi Booking Trip Labuan Bajo merupakan aplikasi Android sederhana bertema travel booking wisata.

Aplikasi ini digunakan untuk:

* melihat daftar destinasi wisata
* melakukan booking trip
* memilih fasilitas tambahan
* memilih tanggal keberangkatan
* menampilkan hasil booking
* share hasil booking

## Fitur Aplikasi

* RecyclerView
* Explicit Intent
* Implicit Intent
* SharedPreferences
* DatePicker
* Form Validation
* Share Booking

## Teknologi Yang Digunakan

* Java
* Android Studio
* RecyclerView
* CardView
* SharedPreferences
